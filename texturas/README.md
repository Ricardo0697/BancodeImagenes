# Texturas

Esta carpeta contiene imágenes de texturas para superficies, materiales y ambientes en juegos 2D y 3D.

## Categorías
- Naturales (madera, piedra, tierra, agua)
- Artificiales (metal, concreto, ladrillo)
- Orgánicas (piel, tela, vegetación)
- Abstractas (patrones, gradientes)

## Formatos sugeridos
- PNG
- JPG
- TGA (con canal alfa)

## Consideraciones
- Texturas seamless (sin costuras) cuando sea posible
- Incluir mapas de normales (normal maps) si aplica
- Resoluciones: 256x256, 512x512, 1024x1024, 2048x2048 px
