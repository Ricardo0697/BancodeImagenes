# BancodeImagenes
El banco contendría imágenes creadas para iconos, miniaturas de Juegos, sprites, texturas y mundos/niveles para otros tipos de juegos.

## Contenido

### Iconos
Imágenes diseñadas para iconos de aplicaciones y juegos.

### Miniaturas
Miniaturas y previsualizaciones para juegos.

### Sprites
Gráficos 2D utilizados para personajes, objetos y animaciones en videojuegos.

### Texturas
Imágenes de texturas para superficies, materiales y ambientes en juegos 2D y 3D.

### Mundos / Niveles
Diseños de mapas, fondos y niveles completos para diferentes tipos de juegos.

## Estructura de Carpetas

```
BancodeImagenes/
├── iconos/          # Iconos para aplicaciones y juegos
├── miniaturas/      # Previsualizaciones de juegos
├── sprites/         # Sprites para personajes y objetos
├── texturas/        # Texturas para materiales y superficies
└── mundos/          # Mundos y niveles de juegos
```

## Uso

Todas las imágenes de este banco están organizadas por categoría para facilitar su uso en proyectos de desarrollo de videojuegos.
